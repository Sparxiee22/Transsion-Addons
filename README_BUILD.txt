- com.kurumidev.transsionaddons.MainActivity

Build in Android Studio:
1. Open this folder as a project.
2. Let Gradle sync.
3. Build > Build APK(s).
4. Install app-debug.apk to your phone.

Build root module zip:
1. Run ./scripts/build-module.sh
2. Flash dist/TranssionAddons-system-module.zip in Magisk/KernelSU/APatch.
3. Reboot.
4. Open Transsion Addons from the module action button or your Settings integration.

Features enabled:
- Rendering Change
- Game Driver Preference
- Display
- Display Debug
- Game Space Controls
- Kernel Manager
- Kernel Settings
- Device Frequency
- Thermal Monitor
- Storage Health
- LED Backlight
- Touch Settings
- Battery & Charging
- Overlay Monitor
- Network Tweaks

Warning:
Disabling thermal protection can overheat or damage the device. Use carefully.
